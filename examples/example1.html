Basic example

An basic example jaws. We make our own canvas, instead of having jaws generate one (or several) for us. We get the canvas-tag from the DOM, fetch its context and send it to Sprite. If you for some reason want to have your own canvas tag this is the way to do it. Add "?debug=1" to the URL and jaws will create a small window with debuginfo.
